# Alzheimer's AI Detection

CNN-based medical imaging system for early Alzheimer's disease diagnosis from brain MRI scans with classification confidence scoring.

## Features

- Multi-class classification: Non-Demented, Very Mild, Mild, Moderate
- Transfer learning with pre-trained CNN architectures
- Grad-CAM visualizations for model interpretability
- Data augmentation pipeline for limited medical datasets
- Model performance evaluation with ROC curves and confusion matrix

## Tech Stack

Python, TensorFlow, Keras, NumPy, Matplotlib, Jupyter

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Alzheimers-AI-Detection.git
cd Alzheimers-AI-Detection
pip install -r requirements.txt
```

### Usage

```bash
jupyter notebook Alzheimers_Detection.ipynb
```
